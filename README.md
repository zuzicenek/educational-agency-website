# educational-agency-website

This project is a simple multi-page website created as a school assignment.

## Project Description
A fictional educational agency offering paid courses for the public.
The goal of the project was to practice HTML5 and CSS3 with semantic elements.

## Features
- 10 HTML pages
- Semantic HTML5 elements (section, article, nav, aside)
- Gallery page
- Contact form
- Embedded video, audio, and canvas
- Responsive layout basics

## Technologies Used
- HTML5
- CSS3
